# CatsEyeStudios

## Installation

npm install

## Running locally

1.  Open first terminal
2.  npm run watch
3.  Open second terminal
4.  node server.js
5.  Open browser to [localhost:1991](http://127.0.0.1:1991/)

## Build

1.  npm run build

## Publish

1.  Push changes to master branch, website will update via Github Pages
